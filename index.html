1. Program to perform arithmetic operations and display the result:

In this program, we start with the initial value of a as 10.

++a increments the value of a by 1 before evaluating the expression, so the value of ++a becomes 11.
a++ increments the value of a by 1 after evaluating the expression, so the value of a++ is still 11, but then the value of a becomes 12.
--a decrements the value of a by 1 before evaluating the expression, so the value of --a becomes 11.
a-- decrements the value of a by 1 after evaluating the expression, so the value of a-- is still 11, but then the value of a becomes 10 again.
The program displays the results at each stage to show the changes in the value of a.
